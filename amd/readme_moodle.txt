TeX2Max Instructions
----------------------
1. Clone https://github.com/KQMATH/tex2max into an unrelated directory.
2. Follow build instructions in the README.md file.
3. Copy /build/tex2max.js to amd/src/tex2max.js
4. Update thirdpartylibs.xml
5. Run `grunt amd`
